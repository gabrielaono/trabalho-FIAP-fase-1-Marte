# Trabalho-FIAP-fase-1-Marte
Projeto de análise de telemetria para missão a Marte

Este é um projeto que faz parte do currículo acadêmico da FIAP. Nele procura-se colocar as aulas em prática, incluindo o uso do Google Colab para a programação em Python. 
Aqui, há uma situação inventada sobre uma viagem à Marte. Assim, com a assistência de IA foi criada a tabela de telemetria sobre a qual o código girou em torno. 

Assim, para acessar o código
1. Faça download da tabela em csv (telemetria_marte)
2. Abra o código
3. Tente usá-lo

É importante considerar que para o código rodar, é preciso ter o Python e a biblioteca Pandas instalados
